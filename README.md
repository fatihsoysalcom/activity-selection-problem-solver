# Activity Selection Problem Solver

This example demonstrates a fundamental engineering problem-solving approach using the Activity Selection Problem. It illustrates how to define a problem, break it down into manageable steps (like sorting by finish time), and apply a greedy strategy to find an optimal solution for maximizing non-overlapping tasks.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
